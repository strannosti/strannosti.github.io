Что такое git merge? git merge это метод для слияние двух веток в единую ветку

Git rebase -- копирует все коммиты из выбранной ветки и копирует из в новую ветку

отличия rebase от merge -- rebase копирует и вставляет все имеющиеся коммиты из ветки в новую ветку, а merge объединяет две ветки в одну

Git fetch — команда которая используется для скачивания изменений из удаленного репозитория в локальный репозиторий. water вода

отличия fetch от pull -- fetch загружает что то без изменения ветки , git pull пытается подогнать ветку под себя

git workflow это шпаршалка по использыванию git

как работает? позволяет всем членам команды вносить изменения непосредственно в основную ветку, при этом каждое изменение комитится
